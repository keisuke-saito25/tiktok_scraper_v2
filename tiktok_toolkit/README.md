# tiktok_toolkit

## インストール

1. **リポジトリのクローン**

   ```bash
   git clone <リポジトリのURL>
   cd tiktok_toolkit
   ```

2. **仮想環境の作成**

   ```bash
   python3 -m venv venv
   ```

3. **仮想環境の有効化**

   ```bash
   source venv/bin/activate
   ```

4. **依存関係のインストール**

   ```bash
   pip install -r requirements.txt
   ```
## 実行方法

- **通常**

    ```bash
    python3 tiktok.py
    ```
